Usage Examples

Learn how to use Flame for your projects

Data Preparation Pipeline

Generate self-contained component code snippets from GitHub repositories:

bash scripts/collect_gh_code_run.sh

Render code snippets to images:

bash scripts/renderer_run.sh

Model Training

Train the Flame model with your prepared dataset:

python model/scripts/train/pretrain_STAGE2.sh

The training process includes three stages: vision encoder pretraining, layout interpretation, and instruction-tuning.

Model Evaluation

Evaluate your trained model on the Flame-React-Eval benchmark:

python evaluator/eval/index.py --eval_only --model_path /path/to/model

This will run automated testing for both functional correctness and visual fidelity.
